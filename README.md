# IbisUI

Ibis is Website UI Toolkit with Dart, like Flutter, AngularDart, React.js, and Next.js etc.

## Usage

A simple usage example:

```dart
import 'package:ibis_ui/ibis_ui.dart';

main() {
  var awesome = new Awesome();
}
```
